# Generador y Validador de Contraseñas Seguras

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un programa en Python que permite generar y validar contraseñas seguras. El sistema analiza si una contraseña cumple con criterios básicos de seguridad, como longitud mínima, uso de letras mayúsculas, letras minúsculas, números y caracteres especiales.

## Objetivo

Desarrollar un programa que ayude al usuario a crear o verificar contraseñas más seguras, aplicando condiciones básicas de validación mediante estructuras de programación.

## Herramientas utilizadas

- Python
- RAPTOR
- GitHub
- Visual Studio Code / IDLE de Python

## Archivos del repositorio

- `generador_contrasenas.py`: contiene el código principal del programa en Python.
- `diagrama_contrasena.rap`: contiene el diagrama de flujo elaborado en RAPTOR.
- `captura_raptor.png`: evidencia visual del diagrama de flujo funcionando.

## Funcionamiento del programa

El programa solicita al usuario ingresar una contraseña. Luego revisa si cumple con los requisitos establecidos:

- Tener una longitud mínima de 8 caracteres.
- Contener letras mayúsculas.
- Contener letras minúsculas.
- Contener números.
- Contener caracteres especiales.

Si la contraseña cumple con los requisitos, el programa indica que es segura. Si no cumple, muestra cuáles condiciones faltan.

## Relación entre RAPTOR y Python

El diagrama realizado en RAPTOR representa la lógica principal del sistema de forma resumida. En RAPTOR se muestra el ingreso de una contraseña, el cálculo de su longitud y la validación básica de seguridad.

El código en Python desarrolla esta lógica de manera más completa, agregando más condiciones de validación para fortalecer la seguridad de la contraseña.
